  ## Desctiption
  ### This is a simple backend for an e-commerce type online store. It allows the user to create new products, categories, and tags as well as update and delete any of those. It allows the user to access that data, which is stored inside a MySQL database. You can view a walkthrough video via this [link](https://youtu.be/Sj9XZki-K8A).
  ## Installation
  ### Clone the GitHub repository and run "npm i". Create the database using the Schema. Run "npm run seed" in the terminal and then start the server using "node server.js"
